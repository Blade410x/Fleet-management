# **Fleet-management**
A command line based fleet management application built with Python and MySQL

# **What it does** :
Allows a bus fleet owner to manage their entire operation through a menu driven interface. Features include adding and removing vehicles, assigning buses to routes, tracking active and inactive fleet and managing finances including running costs, maintenance and profits per vehicle

# **Technologies used :**
- Python 
- MySQL 
- Tabulate Library

# **Features :**
- Vehicle management with number plate tracking 
- Route creation
- Automated status tracking via generated columns
- Finance tracking with net profit calculation per vehicle
- Input validation and error handling throughout

# **How to run :**
Install dependencies with `pip install mysql-connector-python`, `pip install tabulate`, set up MySQL with your credentials in line 3 and run `python fleet_management1.py`.
